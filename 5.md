✅  Готово! Два способа деплоя на выбор

  📦 Что создано:

  1. deploy.sh — автоматический деплой через rsync
  2. PYCHARM_DEPLOYMENT.md — пошаговая инструкция для PyCharm

  ---
  🚀 Способ 1: Быстрый деплой через скрипт
  
  ./deploy.sh root@твой_сервер_ip

  Что делает скрипт:
  - Создаёт папку /opt/learnflow на сервере
  - Загружает все нужные файлы
  - Автоматически исключает .venv, .git, __pycache__
  - Устанавливает права на скрипты 
  - Показывает следующие шаги
  
  Рекомендуется для первой загрузки (быстрее всего)

  ---
  🎯 Способ 2: PyCharm Deployment
  
  Открой: PYCHARM_DEPLOYMENT.md — там полная инструкция

  Кратко:

  1. Tools → Deployment → Configuration
    - Host: твой_ip
    - User: root
    - Root path: /opt/learnflow
  3. Excluded paths: .venv, .git, __pycache__, etc.
  4. ПКМ на проект → Deployment → Upload
  
  Рекомендуется для обновлений (удобнее)

  ---
  📋 Что загружается:
  
  ✅  Загружается:
  - Docker конфиги, nginx, .env.production
  - Весь код (learnflow/, src/)
  - Скрипты, документация 
  
  ❌  Исключается:
  - .venv/, .git/, __pycache__/
  - staticfiles/, media/, logs/
  
  ---
  🎬 После загрузки на сервере:
  
  ssh root@server_ip
  cd /opt/learnflow

  Всё готово к загрузке! Выбирай способ и деплой 🚀
